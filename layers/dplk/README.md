# DPLK Layer

Layer untuk modul Dana Pensiun Lembaga Keuangan (DPLK) - modul terbesar dalam InPension.

## Features

### Participant Management
- Registrasi Peserta (Personal & Company)
- Verifikasi PPIP & DKP
- Membership Management
- Data Change Requests

### Claims
- Retirement Benefit Claims
- Transfer of Funds
- Withdrawing Funds

### Financial
- Invoice & Billing
- Data Cycle
- NAB Record

### Reports
- Various DPLK reports
- Analytics & Statistics
