# Company Layer

Layer untuk portal perusahaan dalam sistem InPension.

## Features

### Company Profile
- View and edit company profile
- Company verification status
- Corporate account management

### Participants Management
- View DKP participants
- View PPIP participants
- Active/Inactive participants
- Participant data updates

### Contribution Management
- Upload contribution data
- Contribution history
- Pending contributions
- Contribution reconciliation

### Claims
- Submit claims
- View claim status
- Claim history
